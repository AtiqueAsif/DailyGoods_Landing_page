# DailyGoods – Furniture Landing Page (practice)

DailyGoods is a modern furniture landing page UI design created to showcase products cleanly and minimally. 

This project focuses on a simple and user-friendly interface for browsing furniture, exploring collections, and discovering products.

The design was created using Figma and Google Stitch.

---

## Project Overview

This is a landing page design for a furniture brand website.

The goal is to present products in a visually clean and organized way so users can easily explore and choose items.

---

## Features

- Clean and modern homepage design  
- Product showcase section  
- Featured collection  
- Wishlist / saved items concept  
- Customer testimonials  
- Email subscription section  
- Simple navigation and footer  

---

## Design Style

- Minimal and clean layout  
- Soft colors and spacing  
- Modern typography  
- Focus on product visuals  

---

## Tools Used

- Figma  
- Google Stitch
